# Archive@Home
一个基于 Telegram Bot 的分布式解析系统，用于解析 E-Hentai 画廊链接，并返回归档下载链接。

## 使用 Cloudflare Workers 部署节点

1. 复制 [_worker.js](https://github.com/taskmgr818/archive-at-home/blob/main/server/_worker.js) 文件中的代码到你的 Workers 控制台代码编辑区  
2. 修改代码中的 `COOKIE` 值
3. 保存并部署
